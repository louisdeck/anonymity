# Anonymity in blockchains

Master thesis on anonymity in blockchains<br/>

4 months internship in LIS (Laboratoire d'Informatique & Systèmes)<br/>

- Phase 1: Categorize the current state of anonymity in blockchains in terms of namespaces
- Phase 2: Analyze methods of achieving anonymity: VPN (IPsec tunnel), Mixer, Tor, stealth address, ring signature
- Phase 3: From an adversary's point of view, measure the anonymity provided by these methods using
    different metrics (Shannon entropy, normalized entropy, min-entropy).
- Phase 4: Draw up a protection level table (yes, no, probabilities)
